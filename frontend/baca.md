Bagian frontend bertanggung jawab terhadap tampilan dan interaksi pengguna dengan sistem. Frontend dikembangkan menggunakan HTML, CSS, dan JavaScript tanpa framework agar lebih mudah dipahami.

Tugas utama frontend meliputi:
- Pembuatan halaman login
- Pembuatan dashboard sesuai peran pengguna (Admin, Pustakawan, Anggota, Kepala Perpustakaan)
- Pembuatan halaman pengelolaan data (buku, anggota, peminjaman)
- Menampilkan data dari backend melalui REST API
- Menyesuaikan tampilan dengan mockup yang telah dirancang

Frontend hanya berfokus pada antarmuka pengguna dan tidak menangani logika bisnis maupun pengelolaan database.
